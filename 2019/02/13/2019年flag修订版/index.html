2019年flag修订版

发表于 2019-02-13

今天是2019年2月1日，2019年已经过完一个月了
今天已经是2019年2月13日，2019年过完了一个半月了。有没有很快？有没有感觉时间很恐怖？
你在元旦时定的2019年计划进展的如何的？
我还没有制定。。。。
我月初制定的计划已经开始打脸了。所以要尽快修订一下。
已经很多年没有写年度总结了，现在直接写新一年的年度计划，总感觉可能写不完整，
那就慢慢添加完善好了。
现在看来，2月初写的已经是不能完成了。

总的来说2019年要完成工作、生活、个人成长、习惯养成四大块内容。

工作方面

  • 半年时间完成授信风控相关知识的学习，成为该领域的中等水平专家
  • 完成上半年的KPI
  • 争取今年年底前实现工资涨幅30%

生活方面

  • 完成婚事
  • 学会做饭，少点外卖
  • 跑一场半马
    - [ ] 练出8块腹肌
  • 每周至少一次锻炼

个人成长方面

- [ ] 看完50本书

  • 看完至少12本书
  • 每周至少一篇文章输出
  • 学会一门小语种
  • 学会钢琴，至少三首钢琴曲

习惯养成

- [ ] 绝对不熬夜

  • 尽量不熬夜
  • 科学防脱
  • 睡前不玩手机
  • 早睡早起
  • 打败拖延症
  • 做个自律的人
