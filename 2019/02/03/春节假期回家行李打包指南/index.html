春节假期回家行李打包指南

发表于 2019-02-03

现在在春运回家的高铁上，还有两个小时下车。感觉自己像霍金一样，歪头斜脑，浑身无力。究其原因，我想肯定是昨天晚上睡的时间太少有关。
昨天晚上收拾回家的行李，一个箱子装不下，还带了两个拉杆箱。感觉需要把这个过程整理下来，提升其他朋友的效率。这篇清单有借鉴印象笔记的推文，在此感谢。

回家前的准备工作

  • 购买往返机票/火车票。现在都需要提前三十天从网上抢票了。如果自己买不到，还可以通过抢票软件（据说现在不好用了）、黄牛等途径买票。
  • 重要文件检查
    • 身份证_护照_驾驶证
    • 机票行程单
    • 个人证件照电子版
  • 当地特产、年货和礼品。过年回家不带点年货、礼品回去，总觉得说不过去，而且回家后还有各种亲戚要去串门。所以昨天下午我去采购了一些。事实上最好是提前半个月置办，这样能买到合适的。

出行前一天

  • 设置工作电子邮件自动回复。因为春节假期大家统一休息，这个必要性不大。但如果是自己休年假而客户、同事在上班，这一条就非常有必要了。
  • 结算水电煤气等账单（充点卡_水卡_煤气卡，还信用卡，假期中都没有心思处理这种事情）。
  • 托管宠物和植物。现在朋友圈很多过年不回家的朋友，会提供有偿的上门喂养、铲屎服务哈哈哈。
  • 准备好现金和信用卡。虽然现在支付宝/微信支付非常普及了，但是随身携带现金的习惯还是要有的。有些地方就是没有办法扫支付宝或者刷卡。
  • 和要和的朋友或者同事分享自己的行程。把自己的目的地以及返程日期告诉对方，这样在节后如果出现什么意外，大家能有个照应。

行李打包

  • 身份证件：身份证_护照_驾驶证/港澳通行证，学生证（如果有）
  • 贴身小包_钱包：携带必要的信用卡_借记卡（以备急需取现）以及现金零钱。
  • 拉杆箱：最好是万向轮的。根据行李的多少，决定用哪个尺寸的。
  • 双肩背包（配行李牌）：放水杯、笔记本电脑、手机充电器、食品、阅读的书籍等。
  • 个人常用药：晕车药、感冒药等。
  • 水杯：人到中年就要常备保温杯和枸杞。
  • 正在阅读的书籍：最好是不要太厚，不然徒增行李重量
  • 干/湿纸巾
  • 雨伞
  • 充气枕/眼罩
  • 食品： 可能需要携带一些食品在路上充饥

要带的衣物包括（具体衣物的数量，要根据假期时间的长短来定了）：

  • 内衣裤
  • 袜子
  • 外套
  • 毛衣
  • 衬衫
  • 裤子
  • 大衣/风衣
  • 围巾
  • 鞋子

洗漱用品包括：

  • 剃须刀：出发前充好电，可以少带一根充电线
  • 洗面奶
  • 爽肤水
  • 乳液/面霜
  • 面膜
  • 润唇膏
  • 隐形眼镜护理液和眼镜盒

电子设备包括：

  • 笔记本电脑
  • 平板电脑iPad：提前下载好视频或者书籍等内容
  • 手机
  • 耳机
  • 充电宝
  • 数码相机
  • 三脚架
  • 各类充电线：出发前一晚上一定要给各类电子设备充好电

出门前的检查

有人总结了“伸手要钱”：身份证、手机、钥匙、钱包。非常到位，只要这四个东西齐备了，旅途安全没有问题。此外，还要检查：

  • 特产、年货和礼品
  • 收好晾晒的衣服
  • 切断电源
  • 收好家里的贵重物品
  • 关好门窗
  • 倒垃圾

最后，一定要提前出门，为堵车、检票等预留出充足的时间。
